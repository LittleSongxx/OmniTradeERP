## 🎯 项目简介

OmniTrade ERP 是一个功能完整的**跨境电商智能ERP系统**，具备：

- 🤖 **4大AI服务**：智能定价、库存预测、AI客服、产品描述生成
- 📦 **8大核心模块**：订单/商品/库存/财务/仓库/供应商/客户/租户
- 🌍 **5大平台对接**：Amazon、eBay、Shopee、Lazada、TikTok
- 🏗️ **微服务架构**：基于 Spring Cloud Alibaba
- 📊 **70+单元测试**：完整的测试覆盖
