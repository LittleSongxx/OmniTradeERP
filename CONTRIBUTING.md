# 🤝 Contributing to OmniTrade ERP

感谢您对 OmniTrade ERP 的关注！我们欢迎所有形式的贡献。

## 🚀 如何开始

1. Fork 本仓库
2. Clone 您的 fork: `git clone https://github.com/YOUR_USERNAME/OmniTradeERP.git`
3. 创建特性分支: `git checkout -b feature/your-feature-name`

## 🛠️ 开发环境要求

- JDK 21+
- Node.js 18+
- Maven 3.9+
- Docker & Docker Compose

## ✅ 代码规范

- 遵循 Google Java Style Guide
- 新增方法必须添加 Javadoc
- 新增功能需附带单元测试
- 提交前运行 `mvn test`

## 📝 提交流程

1. Fork 仓库
2. 创建特性分支
3. 提交代码（遵循 commit 规范）
4. 发起 Pull Request
5. 等待 Code Review

## 🐛 Bug 报告

请提交 Issue，包含：清晰的标题、重现步骤、环境信息。

## 💬 交流

有问题？去 [GitHub Discussions](https://github.com/nplszfl/OmniTradeERP/discussions) 提问
